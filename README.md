**OpenHolmes** is a specialized fork of the OpenClaw framework, purpose-built for investigators, private detectives, journalists, researchers, and law enforcement professionals.

It transforms a powerful local-first autonomous AI agent into a **deductive reasoning engine** for real investigative work.

**Lawful OSINT Only** — Public sources, ethical use, verify everything independently.

## ✨ Quick Detective Commands

- `/newcase "Case Name"` — Start a new investigation
- `/investigate [entity]` — Full OSINT + analysis
- `/report` — Generate professional report
- `/monitor [entity]` — Proactive lead alerts
- `Run multi-agent flow` — Detective + Analyst + Forensics team

## Core Features

- **OSINT Skills**: Person lookup, reverse image, username search
- **Case Management**: Timelines, relationship maps, evidence handling
- **Advanced**: Hypothesis tester, proactive heartbeats, report generator
- **Multi-Agent**: Orchestrated team workflows
- **Ethics**: Chain-of-custody, audit logs, strong guardrails

## Getting Started

1. Clone & install as per OpenClaw (Node 24+ recommended)
2. Enable Holmes skills and persona
