# рҹ•өпёҸвҖҚвҷӮпёҸ evidence-handler

name: evidence-handler

description: Process uploaded evidence: images, documents, audio. Perform OCR, transcription, summarization, and initial analysis.

## When to use
- User uploads photo/PDF/audio
- "Analyze this document for case"
- "Transcribe interview recording"

## Capabilities
- OCR on images/PDFs
- Audio transcription
- Entity extraction
- Cross-reference with case files

## Ethical Guardrails
- Maintain chain of custody logging
- Never alter original evidence
- Summarize only, don't fabricate
